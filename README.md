# CryptoScope — Personal Crypto Trading Decision Support

This project is a personal decision support system for mid-term cryptocurrency trading (BTC/ETH). It is designed to provide data-driven signals based on a multi-layer confluence model (Technical, On-Chain, Sentiment, Macro) with a focus on risk management and capital preservation.

## Strategy Overview

CryptoScope uses a **3-Layer Signal Model**:
1. **Direction (Macro + Trend):** Determines the overall trade bias.
2. **Timing (Momentum + Volume):** Fine-tunes entry and exit points.
3. **Risk Gate:** Ensures every trade follows a strict 2% portfolio risk rule.
